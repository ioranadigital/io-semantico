Cómo embeber el formulario de Informe SEO Gratuito

Copia este código HTML en la página donde quieras que aparezca el formulario (por ejemplo, en tu landing page, homepage o página de servicios): 

<!-- Embeber formulario SEO Agent 2026 -->
<iframe
  src="https://tu-dominio.com/lead-form"
  width="100%"
  height="700"
  frameborder="0"
  style="border-radius: 12px; box-shadow: 0 4px 24px rgba(0,0,0,0.1); border: none;"
  title="Informe SEO Gratuito"
>
</iframe>

Configuración recomendada:

  • Ancho (width): 100% para que se adapte al contenedor
  • Altura (height): 600-800px según el dispositivo
  • URL (src): Cambia "tu-dominio.com" por tu dominio real
  • Estilos opcionales: Personaliza border-radius, box-shadow, etc.

Ejemplo con media queries (responsive):

<style>
  .seo-form-container {
    width: 100%;
    margin: 2rem 0;
  }

  .seo-form-container iframe {
    width: 100%;
    border: none;
    border-radius: 12px;
    box-shadow: 0 4px 24px rgba(0,0,0,0.1);
  }

  @media (max-width: 768px) {
    .seo-form-container iframe {
      height: 700px;
    }
  }

  @media (min-width: 769px) {
    .seo-form-container iframe {
      height: 600px;
    }
  }
</style>

<div class="seo-form-container">
  <iframe
    src="https://tu-dominio.com/lead-form"
    title="Informe SEO Gratuito"
  ></iframe>
</div>

Alternativa: Usar JavaScript para cargar el iframe

Si prefieres más control, puedes cargar el formulario dinámicamente:

<div id="seo-form-placeholder"></div>

<script>
  document.addEventListener('DOMContentLoaded', function() {
    const container = document.getElementById('seo-form-placeholder');
    const iframe = document.createElement('iframe');
    iframe.src = 'https://tu-dominio.com/lead-form';
    iframe.width = '100%';
    iframe.height = '700';
    iframe.style.border = 'none';
    iframe.style.borderRadius = '12px';
    iframe.style.boxShadow = '0 4px 24px rgba(0,0,0,0.1)';
    iframe.title = 'Informe SEO Gratuito';
    container.appendChild(iframe);
  });
</script>

Verificar que el webhook está configurado

Antes de embeber el formulario, asegúrate de que:

  1. Tienes VITE_WEBHOOK_URL configurado en tu .env
  2. Tu instancia de n8n está corriendo y el webhook está activo
  3. El dominio está disponible públicamente (no localhost)
