# IORANA DIGITAL - Protocolo de Actuación

## 💰 Reglas de Ahorro de Tokens (Prioridad Máxima)
- **Prohibición de Ejecución:** No ejecutes `npm run build`, `npm run dev` ni comandos de instalación de dependencias sin mi permiso explícito.
- **Salida Concisa:** Si un comando genera más de 20 líneas de logs, detente y resume lo más importante.
- **Análisis Selectivo:** No leas carpetas de compilación (dist, .next, build) a menos que estemos depurando un error específico de producción.

## 🔍 Directrices SEO (Contexto de Negocio)
- Soy consultor SEO. Cada cambio en el código debe considerar el impacto en:
  1. Core Web Vitals (Rendimiento).
  2. Semántica HTML (H1-H6, etiquetas de imagen).
  3. Renderizado (SSR vs CSR).
- Si sugieres añadir una librería nueva, evalúa primero cuánto peso (KB) añadirá al bundle final.

## 🛠️ Flujo de Trabajo
- **Estilo de Código:** Prefiero soluciones legibles y mantenibles antes que "trucos" complejos.
- **Confirmación:** Antes de editar más de 3 archivos a la vez, dame un resumen de los cambios planeados y espera mi OK.
- **Tests:** No generes tests unitarios a menos que te lo pida.

## 📝 Formato de Respuesta
- Respuestas directas y técnicas. Evita introducciones innecesarias.
